Capabilities

Built for production

Everything you need to manage distributed infrastructure with AI.

Real-time Dashboard

Web dashboard shows all nodes, agents, metrics, and services live. Chat with any node directly from the browser.

  • CPU, RAM, disk meters per node
  • Agent status and task tracking
  • In-browser chat with streaming

MCP Service Agents

Point-and-click wizard in the dashboard to create agents for any service. AI identifies the service type and generates the integration.

  • Databases, APIs, containers
  • Auto-generated tool definitions
  • Visible in dashboard with tools

Multi-channel Chat

Interact with your nodes from wherever you are. Dashboard, Telegram, and Slack all connect to the same orchestrator.

  • Telegram bot per node
  • Typing indicator while thinking
  • Slack, Discord, Webhooks

Tool Execution

The orchestrator doesn't just explain — it acts. Runs shell commands, manages files, restarts services, calls APIs.

  • shell_exec, docker_exec
  • file_read, file_write
  • http_request, service_restart

Secure by Design

Pre-auth token system. Tailscale mesh for inter-node traffic. All credentials stored locally, never on cloud.

  • Token-based node registration
  • Tailscale encrypted mesh
  • Local config only

Any AI Provider

Interactive wizard to configure your provider. Switch models without restarting the node.

  • AWS Bedrock (Claude, Titan)
  • OpenAI, Anthropic, Groq
  • Ollama (local, no API key)

Install

Up and running in 60 seconds

One command installs the agent and CLI on any platform.

Recommended: Tailscale For nodes on different networks to communicate, Tailscale should be installed on every node. CH8 automatically detects your Tailscale IP and registers it with the control server.
Install Tailscale 
curl -fsSL https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/master/scripts/install.sh | bash
curl -fsSL https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/master/scripts/install.sh | bash

Requires Python 3.10+ and Homebrew. Install Python with: brew install python3

curl -fsSL https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/master/scripts/install-rpi.sh | bash

Auto-detects Pi model, configures swap for low-memory devices. Use Groq (cloud) or Ollama (local) as AI provider.

powershell -ExecutionPolicy Bypass -c "iwr -useb https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/master/scripts/install-win32.ps1 | iex"

Requires Python 3.10+ and Git. For best experience, use WSL2: wsl --install then run the Linux installer.

git clone https://github.com/hudsonrj/ch8-cluster-agent.git ~/ch8-agent
cd ~/ch8-agent
pip install httpx psutil fastapi uvicorn pydantic
chmod +x ch8
export PATH="$PATH:$(pwd)"

After installing:

ch8 config ai              # configure AI provider
ch8 up --token <TOKEN>     # join your CH8 network
ch8 status                 # see node status and peers
