Process

One command. Full control.

Install on any machine, configure your AI provider, and connect to your network.

01

Install

One-line installer for Linux and macOS. Works on any machine with Python 3.10+.

02

Configure

Pick your AI provider (Bedrock, OpenAI, Groq, Ollama) and communication channels.

Connect

Join your network with a pre-auth token. Node appears on dashboard immediately.

04

Operate

Chat with any node via Dashboard or Telegram. Execute tasks, monitor services, create agents.

Install

Up and running in 60 seconds

One command installs the agent and CLI on any platform.

Recommended: Tailscale For nodes on different networks to communicate, Tailscale should be installed on every node. CH8 automatically detects your Tailscale IP and registers it with the control server.
Install Tailscale 
curl -fsSL https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/master/scripts/install.sh | bash
curl -fsSL https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/master/scripts/install.sh | bash

Requires Python 3.10+ and Homebrew. Install Python with: brew install python3

curl -fsSL https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/master/scripts/install-rpi.sh | bash

Auto-detects Pi model, configures swap for low-memory devices. Use Groq (cloud) or Ollama (local) as AI provider.

powershell -ExecutionPolicy Bypass -c "iwr -useb https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/master/scripts/install-win32.ps1 | iex"

Requires Python 3.10+ and Git. For best experience, use WSL2: wsl --install then run the Linux installer.

Download APK Get Termux

Option 1 — APK: Download and install the APK directly. Requires "Install from unknown sources" enabled.

Option 2 — Termux: Run inside Termux terminal (from F-Droid, NOT Play Store):

curl -fsSL https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/master/scripts/install-android.sh | bash

Tip: Run termux-wake-lock to keep CH8 running in background.

git clone https://github.com/hudsonrj/ch8-cluster-agent.git ~/ch8-agent
cd ~/ch8-agent
pip install httpx psutil fastapi uvicorn pydantic
chmod +x ch8
export PATH="$PATH:$(pwd)"

After installing:

ch8 config ai              # configure AI provider
ch8 up --token <TOKEN>     # join your CH8 network
ch8 status                 # see node status and peers
