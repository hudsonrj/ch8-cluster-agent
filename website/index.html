Custom Agents

Build your own agents with Python decorators. Plug into the orchestration pipeline instantly.

DecoratorsPlug & playReusable
report_agent.py Create a custom agent in minutes 
# 1. Import and decorate
from ch8 import agent, tool

@agent(name="report-builder", memory=True)
class ReportAgent:

    # 2. Define tools — auto-registered by the orchestrator
    @tool
    def generate(self, data: dict) -> str:
        return self.llm.run(f"Build a report from: {data}")

# 3. Deploy — plug & play into any cluster
ReportAgent().deploy()

Install

Up and running in 60 seconds

One command for any platform. Auto-detects hardware and downloads the optimal model.

Prerequisite: Tailscale For nodes on different networks to communicate, Tailscale must be installed and running on every node. CH8 automatically detects your Tailscale IP and registers it with the control server.
Install Tailscale 
curl -fsSL https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/main/scripts/install.sh | bash
curl -fsSL https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/main/scripts/install.sh | bash
powershell -c "iwr -useb https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/main/scripts/install.ps1 | iex"
curl -fsSL https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/main/scripts/install-rpi.sh | bash

Auto-detects Pi model, configures swap, downloads optimal quantized model.

# Via Termux (install from F-Droid first)
curl -fsSL https://raw.githubusercontent.com/hudsonrj/ch8-cluster-agent/main/android/install-termux.sh | bash
Or install the native app: Download APK

After installing, configure and connect:

ch8 config ai              # configure AI provider (Bedrock/OpenAI/Ollama)
ch8 up --token <TOKEN>     # join your CH8 network
ch8 status                 # see node status and peers
